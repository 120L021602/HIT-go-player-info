功能：新增棋手
url:/api/add_player
请求方法：POST
请求参数：
{
name:string;
gender:string;
yearOfBirth:string;
school:string;
department:string;
major:string;
level:string;
nativePlace:string;
yhAccount:string;
comment:string
}
响应：
{
success:boolean;
msg:string;
}

功能：查询棋手个人信息
url：/api/query_player_info
请求方法：POST
请求参数：
{
name:string；
}
响应：
{
success:boolean;
msg:string;
List	
	<
	name:string;
	gender:string;
	yearOfBirth:string;
	school:string;
	department:string;
	major:string;
	level:string;
	nativePlace:string;
	yhAccount:string;
	comment:string
	>
}
